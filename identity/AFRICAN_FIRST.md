# SevenOS African First

SevenOS is African first by product philosophy, not by decoration.

The goal is to build an operating system that treats digital sovereignty,
creation, protection, transmission, community and resilience as system
principles. Visual identity supports that philosophy, but it does not replace
it.

## Core Idea

SevenOS should feel like:

```text
an African modern operating ecosystem for creating, securing, learning,
deploying and preserving digital work.
```

It should not feel like:

```text
Arch Linux with African patterns added on top.
```

## Product Principles

| Principle | Meaning In SevenOS |
| --- | --- |
| Sovereignty | The system explains itself, can be repaired, and avoids hidden dependency on closed platforms. |
| Transmission | Knowledge is surfaced through Griot-style documentation, onboarding and readable commands. |
| Creation | Studio and Forge are first-class work modes, not optional app lists. |
| Protection | Shield makes security visible, guided and respectful of user trust. |
| Community | Profiles, regional accent packs and future SevenStore are designed for shared extension. |
| Resilience | The system is observable through JSON contracts, repair plans, backups and migration paths. |

## Daily Cultural Signal

SevenOS can show a rotating African country signal in terminal and future Hub
surfaces:

- flag
- country name
- capital
- population
- regional accent pack in the future

This is meant to be informative and respectful. It should not turn the UI into
a collage of flags.

## Visual Rules

- Use African geometry as structure: separators, status marks, profile marks,
  onboarding rhythm and layout.
- Keep background pattern opacity low.
- Prefer profile-specific accents over random decoration.
- Do not use flags as the main UI language.
- Do not use sacred or culturally specific symbols without context.
- Do not make SevenOS look like generic cyberpunk, gaming rice or festival
  branding.

## Regional Accent Packs

Regional packs are optional future layers:

- Pan-African
- West Africa
- North Africa
- Central Africa
- East Africa
- Southern Africa
- Diaspora

Each pack may adjust:

- wallpaper
- accent colors
- pattern rhythm
- terminal country signal
- lock screen signal
- Hub welcome tone

The default SevenOS identity remains broad, modern and African first.

The machine-readable source for these packs is:

```text
identity/accent-packs.json
```

Use:

```bash
seven identity packs
seven identity packs --json
```
