# Baobab Inclusive Language Layer

This directory is reserved for Baobab culture and learning catalogs.

SevenOS currently ships English and French as the supported system UI languages.
Baobab-specific catalogs should add African language learning content, oral
knowledge labels, TTS hints and culturally grounded vocabulary without making
Baobab depend on Forge, Shield, Horizon, Pulse or Studio.

Future catalog candidates:

- `wo.json`
- `sw.json`
- `yo.json`
- `am.json`
- `ha.json`
- `bm.json`
