# Baobab Inclusive Language Layer

This directory is reserved for Baobab cultural OS catalogs.

SevenOS currently ships English and French as the supported system UI languages.
Baobab-specific catalogs should add African language learning content, oral
knowledge labels, heritage copy, sound/map/fashion/food/wisdom vocabulary, TTS
hints and culturally grounded UI without making Baobab depend on Forge DevOps,
Shield, Pulse or Studio.

Baobab language should follow the Rooted Technology direction:

- speak of memory, roots, branches, transmission, community review and learning;
- keep French-first labels calm, public and educational;
- avoid exoticizing words, safari/postcard metaphors and decorative cultural
  shortcuts;
- make the AI voice a guide or narrator, not a robot persona;
- mark unreviewed cultural or language material as draft or needing local
  validation.
