# SevenOS Symbol

## Recommended Symbol: The Seven Prism

SevenOS needs a symbol that can be named as easily as Apple, Windows, Linux or
Android:

| System | Public symbol |
| --- | --- |
| macOS | apple |
| Windows | window |
| Linux | penguin |
| Android | robot |
| SevenOS | prism |

The recommended SevenOS symbol is **the Seven Prism**.

## Why A Prism

A prism fits SevenOS better than a literal `7` because it expresses the product
idea, not only the name.

- It turns one source into multiple paths, like SevenOS turning one Linux base
  into several mini OS profiles.
- It belongs naturally to the glass, light, blur and depth language already used
  by SevenOS.
- It suggests intelligence, focus and transformation without becoming a robot or
  a generic AI spark.
- It can scale from a small monochrome glyph to a premium animated boot mark.
- It can contain seven facets or seven light paths without forcing the number
  seven into every logo.

## Symbol Meaning

The Seven Prism represents:

- one system, many focused worlds;
- light passing through glass;
- contextual adaptation;
- profile separation without fragmentation;
- Beyond the Desktop.

## Visual Rules

The symbol should be:

- simple enough to draw from memory;
- recognizable at 16 px;
- usable as a monochrome glyph;
- independent from text;
- distinct from Apple, Windows, Linux, Android, GNOME, KDE and Arch marks;
- calm and premium, not gaming RGB;
- geometric, not mascot-based.

## Core Shape

The core mark is a faceted prism/aperture:

- outer shape: rounded triangular prism or compact heptagonal crystal;
- internal structure: seven facets or seven light lanes;
- primary colors: Seven Blue, Seven Violet and Seven Cyan;
- optional security accent: Seven Green only for Shield/cyber state;
- background: Deep Void for app icons and boot surfaces;
- monochrome fallback: single filled prism silhouette.

## Candidate Symbols Considered

| Candidate | Strength | Weakness | Decision |
| --- | --- | --- | --- |
| Number 7 | direct name link | too literal, not OS-like enough | keep as secondary |
| Window/portal | matches "Beyond the Desktop" | too close to Windows and generic UI | reject |
| Star/septagram | seven-point meaning | can feel occult, flag-like or decorative | reject |
| Crystal | premium and glass-like | less action-oriented than prism | merge into prism |
| Compass | guidance and navigation | less tied to mini OS/light/glass | secondary motif |
| Prism | light, glass, seven facets, adaptive profiles | needs a disciplined shape | recommended |

## Usage Model

- App icon / launcher: prism on Deep Void.
- Boot splash: prism light lanes animate in sequence.
- Settings/About: prism with the SevenOS wordmark.
- Mini OS transitions: prism facets shift accent by profile.
- Shield mode: prism gains a Seven Green trust lane.
- Monochrome contexts: one-line prism silhouette.

`symbol-seven-prism.svg` is the premium icon form. `symbol-seven-prism-mark.svg`
is the transparent mark for bars, badges and compact surfaces. The Waybar

stays stable even when SVG image rendering is unavailable.
