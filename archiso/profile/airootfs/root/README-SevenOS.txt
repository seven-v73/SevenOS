SevenOS Live ISO
African first Linux environment

This live environment contains the SevenOS repository at:

  /opt/SevenOS

Useful commands:

  sevenos-welcome
  cd /opt/SevenOS
  ./install.sh doctor
  ./install.sh status
  ./install.sh base --dry-run

This ISO profile is an early Phase 3 foundation. It is not yet the final SevenOS installer.
