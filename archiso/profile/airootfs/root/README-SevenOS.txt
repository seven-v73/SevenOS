SevenOS Live ISO
African first Linux environment

This live environment contains the SevenOS repository at:

  /opt/SevenOS

Useful commands:

  sevenos-welcome
  seven status
  seven doctor
  sevenpkg meta
  sevenpkg status
  seven-power
  sevenosctl status
  sevenosctl doctor
  cd /opt/SevenOS
  ./install.sh doctor
  ./install.sh status
  ./install.sh installer-plan
  ./install.sh base --dry-run

This ISO profile is an early Phase 3 foundation. It is not yet the final SevenOS installer.
